VibgyorSeek Monitoring Client - How to Run Without Console Window
================================================================================

PROBLEM: Console window keeps appearing? 

SOLUTION: Follow these 2 simple steps!

================================================================================
STEP 1: REBUILD THE APPLICATION
================================================================================

Double-click this file:
    rebuild_no_console.bat

Wait for it to complete. You should see "SUCCESS!" message.

================================================================================
STEP 2: RUN THE APPLICATION CORRECTLY
================================================================================

Double-click this file:
    start_hidden.vbs

That's it! The application will start with:
    ✓ NO console window
    ✓ System tray icon visible
    ✓ GUI dialogs working
    ✓ Logging to logs/log.txt

================================================================================
IMPORTANT: DO NOT RUN FROM COMMAND PROMPT!
================================================================================

If you run the .exe file from Command Prompt or PowerShell, you WILL see
a console window. This is normal Windows behavior.

ALWAYS use start_hidden.vbs to launch the application!

================================================================================
AUTO-START ON WINDOWS LOGIN
================================================================================

1. Right-click start_hidden.vbs
2. Click "Create shortcut"
3. Press Win + R, type: shell:startup
4. Press Enter
5. Copy the shortcut to the Startup folder

Done! The application will start automatically on login with no console.
